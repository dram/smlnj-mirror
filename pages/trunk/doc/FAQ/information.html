SML Information Sources

The SML/NJ web pages contain pointers to the current software distribution for the official public release (version 0.93), and the latest developmental, or "working", versions. There is also a brief description of the revised SML '97 language definition, and pointers to the documentation for the new SML Basis library. The CM Manual, describes CM, the compilation manager for SML/NJ developed by Matthias Blume. The manuals for CM, ml-yacc, ml-lex, and ml-burg can currently be found in the corresponding src directories in the SML/NJ software distribution (e.g. src/cm/Doc/manual.ps, src/ml-yacc/doc/mlyacc.tex).

There is a page comparing the top level environments of SML/NJ 0.93 with current working versions of SML/NJ that comply with the SML '97 basis. This guide is useful for those converting programs to the new basis, or for those learning SML from textbooks that assume the 0.93 environment.

The comp.lang.ml newsgroup carries articles discussing ML in general, including Standard ML and other dialects like Caml and Ocaml. The comp.lang.ml FAQ provides general information about various ML systems and resources.
