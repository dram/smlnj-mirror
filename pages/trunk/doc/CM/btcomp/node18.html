5.1 Anchoring requirement

During bootstrap compilation it is necessary that all pathnames be either anchored or relative to another anchored pathname. This rule guarantees that every path name can be mapped to different locations in the filesystem at different times. (Technically, this restriction is necessary only for library description files.)
