A..6 EBNF for library descriptions

The following set of rules defines the syntax for library descriptions ( library). Notice that although the syntax used for version is the same as that for stdpn, actual version strings will undergo further analysis according to the rules given in section 5:

libkw library   Library   LIBRARY
version stdpn
privilege stdpn
lprivspec { privilege   ( { privilege } ) }
library [ lprivspec] libkw [[( version )] elst] (is   IS) members
