G. Exports of library $smlnj/cm/tools.cm

As described in section 11, it is possible to extend CM's set of available tools using the programming interface provided by structure Tools. This structure--together with its corresponding signature TOOLS--is exported by library $smlnj/cm/tools.cm. The same library also exports structure Sharing, structure Version, and a corresponding signature VERSION.
