The Standard ML Basis Library

We are releasing this draft version of the SML Basis Library specification so that the SML community may comment on it. This document is the result of three years of work by a small group of SML implementers. The latest versions of Moscow ML and Standard ML of New Jersey, as well as Harlequin's MLWorks, implement significant portions of this new library.

The document is generated from ML-Doc sources (ML-Doc is an SGML DTD for documenting SML interfaces). There are still some formatting bugs in the generating tools, which we apologize for in advance.

The Basis is available in several different forms:

  • The Standard ML Basis Library Manual (in HTML).
  • The Standard ML Basis Library Manual (in HTML) mirrored in Denmark.
  • A compressed tar file of the HTML pages.
  • A compressed tar file of the SML signatures.
