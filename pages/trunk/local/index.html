SML/NJ Chicago Developers Site

This is a password protected web site serving the local Hyde Park SML/NJ development community (and potentially all active SML/NJ developers). The source files for this web site are found in the directory /stage/web_static/smlnj/htdocs/local on UofC CS Dept servers. Feel free to edit these pages to add or correct information, but make sure any new or changed files are group sml and group writable.

We are transitioning the SML/NJ project from sourceforge to a local gforge server. This page will provide documentation on how to use the local vacilities.

Developer Blog
There is a SML/NJ developer blog at http://smlnj.blogspot.com/. You can join this by establishing a Google account and sending George Kuan a request to joint with your Google id.

Mailing List
The smlnj-hp mailing list address is smlnj-hp at mailman.cs.uchicago.edu. This mailing list is a backup to the Developer Blog, and you should primarily use the developer blog for local communication. We can arrange for top level posts to the blog to be emailed to the smlnj-hp mailing list.

This is a members-only mailing list. Email from anyone who is not subscribed to the list will be discarded. If you might sometimes send email to the list from an alternate email account, please send Dave MacQueen your alternate email address for inclusion in the whitelist.
