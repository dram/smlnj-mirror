Updating smlnj web pages

The smlnj web site is maintained as an svn repository with URL 
   https://smlnj-gforge.cs.uchicago.edu/svn/smlnj/pages/trunk
The actual server page sources are located at 
   /stage/web_static/smlnj/htdocs
on the UofC servers. This is a working directory checked out from the repository mentioned above.

If you want to edit the smlnj pages, you should check out your own working copy of the pages/trunk project, make your edits in your working copy, and then commit them to the svn repository.

Your changes can then be made visible at the main www.smlnj.org URL, which is an alias for smlnj.cs.uchicago.edu, by doing an svn update in the htdocs directory mentioned above.

NOTE: Certain distribution-specific files are not in the repository. In particular, dist/working/110.xx/NOTES, which contains some documentation files like MACOSXINSTALL. These files have to be edited directly in the corresponding site source directories.

This page and other local developer documentation are found in the local subdirectory of the pages directory. This local site is password protected (user: smlnj, password: semantix), and there are no links to it from the main smlnj web site.

Formerly the web server for www.smlnj.org was at sourceforge, and the directory containing the web pages was located at: 
   /home/groups/s/sm/smlnj/htdocs
But this old version is no longer maintained.
