The SML/NJ Compilation Manager (CM)

There are two different versions of the compilation manager.
  1. The old version (.ps) that is still used in the official release version of SML/NJ.
  2. A new version (.ps) that has been substantially revised, was rewritten from scratch, and for some time now (since version 110.20) has been included with working versions of SML/NJ.

There is also some documentation for the (new) bootstrap compiler (.ps).
