Improved Backends

Some of the instruction selection modules have been rewritten to take better advantage of the instruction sets of the target architectures. The main improvements can be classified as follows:
  • Some architectures provide indexing modes with scaling. These addressing modes are now used.
  • 64-bit instructions can now be specified using the new MLTREE languagge.
  • Strength reduction is performed on multiplication and division by constants by all new instruction selection modules.
Specific changes for each architectures are listed in the logs below:

  • PA-RISC
  • Alpha
