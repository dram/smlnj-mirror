Instruction Selection Modules

All instruction selection modules satisfy the MLTREECOMP signature, which is defined as follows: 
signature MLTREECOMP = sig
    structure T : MLTREE
    structure I : INSTRUCTIONS

    val selectInstructions : 
        (I.instruction,T.mlrisc list,I.C.regmap) T.stream ->
        (T.stm,T.mlrisc list,I.C.regmap) T.stream 
end
Intuitively, this signature states that the instruction selection module transforms a stream of MLTREE statements into a stream of instructions of the target machine.

  • PA-RISC
