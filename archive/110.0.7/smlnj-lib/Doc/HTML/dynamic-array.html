The DynamicArrayFn functor

Synopsis

functor DynamicArrayFn (MONO_ARRAY) : MONO_DYNAMIC_ARRAY

The DynamicArrayFn functor takes a monomorphic array structure as an argument. It produces a structure implementing dynamic arrays. These act like arrays of unbounded length.
