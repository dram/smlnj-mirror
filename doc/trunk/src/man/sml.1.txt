SML(1)
======
:doctype: manpage

NAME
----
sml - an interactive Standard ML system

SYNOPSIS
--------
*sml* ['OPTIONS']

DESCRIPTION
-----------
It is an incremental, interactive compiler that accepts declarations and expressions
to be evaluated from standard input.
