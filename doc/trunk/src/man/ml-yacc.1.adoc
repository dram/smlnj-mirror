NAME
----
ml-yacc - an LALR(1) parser generator

SYNOPSIS
--------
*ml-yacc* 'file'

DESCRIPTION
-----------

ML-Yacc is an LALR(k) parser generator for *Standard ML* that is modeled
on the YACC parser generator for the *C* language.

AUTHOR
------
ml-yacc(1) was written by David Tarditi.
