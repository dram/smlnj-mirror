== Usage

For https://smlnj.org[*SML/NJ*], include `$/regexp-lib.cm` in your
*CM* file.

For use in https://www.mlton.org/[*MLton*], include
`$(SML_LIB)/smlnj-lib/RegExp/regexp-lib.mlb` in your *MLB* file.
