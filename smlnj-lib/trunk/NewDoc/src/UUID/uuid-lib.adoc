== Usage

For https://smlnj.org[*SML/NJ*], include `$/uuid-lib.cm` in your
*CM* file.

For use in http://www.mlton.org/[*MLton*], include
`$(SML_LIB)/smlnj-lib/UUID/uuid-lib.mlb` in your *MLB* file.
